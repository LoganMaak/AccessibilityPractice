Summary of updates:

  • Changed background color to be a much lighter green, enabling the dark grey text to be more easily read (contrast ratio of 13.14:1).
  • Changed the basic div tags to nav tags, as well as deleting the br tags and adding p tags to make the content more screen reader friendly.
  • Changed the css selectors from id selected elements to h1, h2 etc.
  • Added alt atributes to the images to make them screen reader friendly.
  • Added a custom buttons to the audio player to support screen readers and older browsers.
  • Added the role attribute to the search form, as well as the aria-attribute to the search input box.
  • Added the role attribut to the comment form, as well as aria-attribute to both input boxes, as well as adding the label tag to the text labels.
  • Added the tabindex attribut, set to 0, to the show comment buttons, as well as adding the needed javascipt to make the button function when Enter is pressed while it has focus.
  • Added a caption to the table describing what information is being presented, changed all headings from td to th tags as well as indicating whether it was a row or column using the scope attribute.
  • Changed the font size and weight of the overall text to be more legible for the vision impared.
